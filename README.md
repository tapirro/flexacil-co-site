# Flexacil Colombia

> Suplemento articular con cúrcuma, glucosamina, condroitina y uña de gato para la salud de las articulaciones.

**[Visitar flexacil.co](https://flexacil.co)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Flexacil Colombia](https://flexacil.co), un recurso de salud basado en evidencia científica enfocado en **Articulaciones**.

### Ingredientes clave

Curcumina (95% curcuminoides), sulfato de glucosamina, condroitina, colágeno tipo II, uña de gato (*Uncaria tomentosa*)

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
